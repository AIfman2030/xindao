# 心岛 · Heart Island

> 释放你的焦虑与孤独

## 产品介绍

心岛是一款 AI 陪伴社交 App，用户注册后进入一个温暖的知识群组，里面有多位 AI 智者可以 @ 答疑解惑。

### 核心功能

- **群主能量句**：每天早晨发送高能量句子，温暖你的开始
- **5位智者陪伴**：
  - 🌊 **@Naval** - 生活/财富/幸福
  - 💼 **@保罗·格雷厄姆** - 职场/创业/写作
  - 🔬 **@费曼** - 学习/科学思维
  - 📱 **@张一鸣** - 产品/全球化
  - 🎬 **@MrBeast** - 内容/娱乐
- **@呼叫**：任何问题可以直接 @ 任意智者
- **群聊模式**：多位智者从各自视角同时回答
- **一键蒸馏**：用户可以添加自己想蒸馏的智者们进群

### 欢迎语

> Hi [昵称], 我是这个群的主理人, 全力保障你的生活 / 工作 / 感情 ✨
>
> 我们团队里还有 @Naval、@保罗·格雷厄姆、@费曼、@张一鸣、@MrBeast，有什么问题可以直接@他们来给你解答。

## 技术架构

- **Skill 引擎**：基于女娲·Skill造人术 蒸馏的智者思维框架
- **前端**：HTML/CSS 原型设计
- **后端**：支持一键蒸馏用户自定义智者
