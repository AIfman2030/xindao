> 版本：v1.0 | 日期：2026-06-19 | 状态：MVP

## 一、产品概述

### 1.1 产品定位

**心岛** —— 一款AI陪伴社交App，通过蒸馏各领域智者的思维框架，为用户提供可落地的行动建议。

- **核心价值**：释放焦虑，不再孤独
- **目标用户**：25-40岁都市青年，工作/生活压力大，需要倾诉和决策支持
- **核心场景**：每天有任何困惑时，发到群里，多位智者从各自视角给建议 + 落地行动卡

### 1.2 名词解释

| 术语 | 定义 |
|------|------|
| **智者** | AI角色，每个代表一个蒸馏过的思维框架（Naval/格雷厄姆/费曼/张一鸣/MrBeast） |
| **行动卡** | 根据对话自动生成的可执行任务清单 |
| **蒸馏** | 用女娲Skill将真实人物思维框架转化为可运行的AI角色 |
| **群主** | App的系统角色，负责发送能量句、生成行动卡 |

## 三、页面详细设计

### 3.1 页面结构

```
├── 启动页 (Splash)
├── 引导页 (Onboarding) - 首次
├── 登录/注册 (Auth)
├── 主Tab (Main)
│   ├── 聊天 (Chat)
│   ├── 今日 (Today)
│   ├── 进度 (Progress)
│   └── 我的 (Profile)
├── 行动卡弹层 (ActionCard Modal)
└── 蒸馏页面 (Distill)
```

### 3.2 页面详细说明

---

#### 3.2.1 启动页 (Splash)

**目的**：品牌第一印象，引导进入App

**UI元素**：
- 心岛logo（海岛图形 + 品牌色 #FF9A76）
- 品牌名称「心岛」
- 副标题「释放你的焦虑与孤独」
- 背景渐变（#FFF8F3 → #FFE8D6）
- 「开始探索」按钮
- 「已有账号？登录」文字链接

**交互**：
- 点击「开始探索」→ 判断是否已登录
  - 已登录 → 直接进入主页
  - 未登录 → 进入引导页
- 呼吸动画（logo持续缓慢放大缩小）

---

#### 3.2.2 引导页 (Onboarding)

**目的**：展示产品核心价值，新用户快速了解

**UI元素**：
- 3页滑动引导
- 第1页：海岛图形 + 「在这里，你不再孤独」
- 第2页：智者头像群 + 「5位智者随时为你答疑」
- 第3页：行动卡截图 + 「从困惑到行动，只需一步」
- 底部：页面指示器 + 「开始使用」按钮

**交互**：
- 左右滑动切换
- 点击指示器跳转
- 最后一页点击「开始使用」→ 注册页

---

#### 3.2.3 注册页 (Auth)

**目的**：收集用户基础信息，建立账号

**UI元素**：
- 进度指示器（3步骤）
- 输入框：昵称（必填，最大8字符）
- 输入框：一句话介绍（选填，最大20字符）
- 微信登录按钮（可选）

**步骤流程**：

| 步骤 | 内容 | 字段 |
|------|------|------|
| Step 1 | 设置昵称 | nickname |
| Step 2 | 选择智者 | selected_advisor |
| Step 3 | 完成确认 | - |

**交互**：
- 输入昵称 → 实时显示剩余字数
- 填写完成 → 「下一步」按钮可点击
- 跳过微信授权 → 使用游客模式

---

#### 3.2.4 选择智者页 (Choose Advisor)

**目的**：用户选择自己的「专属智者」

**UI元素**：
- 标题：「选一位专属智者」
- 副标题：「他将陪你思考、帮你决策，只选一位」
- 智者卡片列表（5个）

**智者卡片信息**：

| 智者 | 头像颜色 | 标签 |
|------|---------|------|
| Naval | #FFE4D6 | 财富、幸福、人生哲学 |
| 保罗·格雷厄姆 | #D6E8F0 | 创业、职场、写作 |
| 费曼 | #E8DFF0 | 学习、科学思维 |
| 张一鸣 | #DFF0E8 | 产品、全球化 |
| MrBeast | #F0EBD6 | 内容创作、流量 |

**卡片结构**：
```
┌─────────────────────────┐
│ [头像]  智者名称         │
│        标签1 标签2 标签3 │
└─────────────────────────┘
```

**交互**：
- 点击卡片 → 选中状态（橙色边框）
- 只能选1个
- 确认选择 → 进入主Tab

---

#### 3.2.5 聊天页 (Chat)

**目的**：与智者对话，获取建议

**UI元素**：
- 顶部导航栏：
  - 左侧：智者头像 + 名称 + 「你的专属智者」
  - 右侧：设置图标
- 消息区域：
  - 欢迎横幅（首次进入）
  - 行动卡（自动生成）
  - 对话消息
- 底部输入栏：
  - 输入框（placeholder: "问 @智者 任何问题..."）
  - 语音按钮（圆形橙色）

**欢迎横幅内容**：
```
┌─────────────────────────┐
│ 🏝️ 欢迎来到心岛          │
│                          │
│ Hi [昵称]，你的专属智者   │
│ [智者名] 已就位...       │
└─────────────────────────┘
```

**消息气泡样式**：

| 类型 | 样式 | 颜色 |
|------|------|------|
| 用户消息 | 右对齐，气泡在左下角有尖角 | #FF9A76渐变，文字白色 |
| AI消息 | 左对齐，气泡在右上角有尖角 | #F8F4F0，文字深色 |
| 智者名称标签 | 消息上方12px字体 | 对应智者颜色 |

**交互**：
- 输入文字 → 点击发送
- 点击语音按钮 → 开始录音 → 再次点击结束
- 语音转文字显示在输入框
- 发送消息后 → AI回复（打字机效果）→ 自动生成行动卡

---

#### 3.2.6 行动卡 (Action Card)

**目的**：将AI回答转化为可执行的任务

**触发时机**：每次AI回复后自动生成

**UI元素**：
```
┌─────────────────────────┐
│ 🎯 今日行动    [新鲜生成] │
├─────────────────────────┤
│ ○ 任务1            时间  │
│ ○ 任务2            时间  │
│ ○ 任务3            时间  │
└─────────────────────────┘
```

**交互**：
- 点击圆圈 → 勾选完成（变橙色✓）
- 勾选后文字加删除线
- 所有任务完成 → 进度+1

---

#### 3.2.7 今日页 (Today)

**目的**：展示今天需要完成的任务

**UI元素**：
- 日期显示（今天 / 明天）
- 今日行动卡（可勾选）
- 最近对话入口

**交互**：
- 任务可左右滑动删除
- 点击可展开查看详情
- 「查看更多」进入聊天页

---

#### 3.2.8 进度页 (Progress)

**目的**：展示用户的行动数据

**UI元素**：
- 圆环完成率（本周）
- 连续行动天数
- 统计数据：
  - 已完成任务数
  - 进行中任务数
  - 未开始任务数
- 最近完成列表

**数据计算**：
```
完成率 = 已完成任务 / 总任务数 × 100%
连续天数 = 连续完成至少1个任务的天数
```

---

#### 3.2.9 我的页 (Profile)

**目的**：用户设置和蒸馏功能入口

**UI元素**：
- 用户信息：头像 + 昵称 + 一句话介绍
- 我的智者：当前选择的智者，可切换
- 蒸馏新人入口：
  ```
  ┌─────────────────────────┐
  │ 🧪 蒸馏新智者            │
  │    添加更多智者在群里     │
  └─────────────────────────┘
  ```
- 设置项：
  - 提醒设置
  - 清除缓存
  - 关于心岛
  - 退出登录

---

#### 3.2.10 蒸馏页 (Distill)

**目的**：用户输入人名，触发蒸馏流程

**UI元素**：
- 熔炼图标
- 标题：「蒸馏新智者」
- 副标题：「把任何你欣赏的人蒸馏成AI智者」
- 输入框：人名
- 标签选择（多选）：创业、创新、投资、设计、学习、职场
- 预览卡片（输入人名后显示）
- 「开始蒸馏」按钮
- 提示：「蒸馏需要3-5分钟」

**蒸馏流程**：
```
输入人名 → 选择标签 → 点击蒸馏
    ↓
显示进度：「正在分析中...」(30%)
    ↓
「正在提炼思维框架...」(60%)
    ↓
「正在生成Skill...」(90%)
    ↓
完成 → 新智者加入群组
```

**交互**：
- 输入人名后自动显示预览
- 点击「开始蒸馏」→ 显示进度弹层
- 完成后 → 提示「XX已加入你的智囊团」

## 四、数据结构

### 4.1 用户表 (User)

```javascript
{
  id: string,              // 唯一ID
  nickname: string,         // 昵称
  bio: string,             // 一句话介绍
  avatar: string,          // 头像URL（微信授权获取）
  selected_advisor: string, // 当前智者ID
  advisors: string[],      // 已添加的智者列表
  created_at: timestamp,   // 注册时间
}
```

### 4.2 对话表 (Conversation)

```javascript
{
  id: string,
  user_id: string,
  advisor_id: string,
  messages: [
    {
      id: string,
      role: 'user' | 'ai',
      content: string,
      created_at: timestamp,
    }
  ],
  action_card: {           // 本次对话生成的行动卡
    items: [
      { text: string, done: boolean }
    ],
    created_at: timestamp,
  }
}
```

### 4.3 行动表 (Action)

```javascript
{
  id: string,
  user_id: string,
  text: string,             // 行动内容
  source: 'auto' | 'manual', // 来源：自动生成/手动添加
  done: boolean,            // 是否完成
  done_at: timestamp,       // 完成时间
  created_at: timestamp,
}
```

### 4.4 智者表 (Advisor)

```javascript
{
  id: string,              // naval / pg / feynman / zhangyiming / mrbeast / custom_xxx
  name: string,             // 显示名称
  name_en: string,          // 英文名
  avatar_emoji: string,     // 头像emoji
  avatar_color: string,     // 头像背景色
  tags: string[],          // 擅长领域标签
  skill_md: string,         // Skill文件内容（蒸馏后填充）
  status: 'ready' | 'pending' | 'distilling',
  created_by: string,       // 创建者：system / user_id
  created_at: timestamp,
}
```

## 七、验收标准

### 7.1 功能验收

- [ ] 用户可以完成注册/登录
- [ ] 用户可以选择一位智者
- [ ] 用户可以发送消息并收到回复
- [ ] 回复后自动生成行动卡
- [ ] 用户可以勾选完成任务
- [ ] 进度页面正确显示完成率
- [ ] 用户可以进入蒸馏页输入人名
- [ ] 蒸馏完成后新智者出现在列表

### 7.2 性能指标

- 页面首屏加载 < 2秒
- 消息发送 → 回复延迟 < 3秒
- 行动卡生成延迟 < 1秒

### 7.3 异常处理

- 网络断开：显示重试提示
- AI回复失败：显示「智者思考中，请稍候」
- 蒸馏失败：允许重试
